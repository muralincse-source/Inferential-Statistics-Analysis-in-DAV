# Inferential Statistics Analysis in DAV

## Overview
This project demonstrates Inferential Statistics concepts using Python in Data Analytics & Visualization (DAV).

The notebook performs statistical analysis to draw conclusions and make predictions from data samples.

## Features
- Data loading and preprocessing
- Statistical analysis
- Hypothesis testing
- Probability analysis
- Data visualization
- Interpretation of statistical results

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SciPy
